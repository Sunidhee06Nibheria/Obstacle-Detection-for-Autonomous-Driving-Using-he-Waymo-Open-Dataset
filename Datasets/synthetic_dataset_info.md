# Synthetic Dataset Information

## Dataset Type
Synthetic Waymo-style multi-modal dataset.

## Sensor Modalities
- RGB Camera Images
- LiDAR Point Cloud Data
- Segmentation Masks
- Bounding Box Annotations

## Classes Used
The dataset contains the following object classes:

1. Car
2. Pedestrian
3. Cyclist
4. Truck
5. Bus
6. Motorcycle
7. Traffic Cone
8. Barrier

Total Classes: 8

## Weather Conditions Simulated
The dataset includes simulated environmental conditions:

- Clear
- Rain
- Fog
- Night

These conditions improve model robustness.

## Number of Samples

Training Samples: 500–1000  
Validation Samples: 100–200  
Test Samples: 100–200  

(Note: Synthetic data generated for experimental purposes.)

## Dataset Purpose

The dataset is designed to simulate real-world autonomous driving scenarios and test multi-modal sensor fusion models under varying environmental conditions.
